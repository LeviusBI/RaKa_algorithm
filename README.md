#### Task description
```
Implement Rabin-Karp's algorithm to substring P alignment to string T. Input format - FASTA. Ouput format - locus of P in T.
Example:

Input: P.fa, T.fa
Output: Alignment locus

Sample_input:
>P
ATC
>T
AATCCG
Sample_output:
2
```
